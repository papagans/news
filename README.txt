Приложение написано таким образом, что только у юзера ADMIN есть функция CRUDа для сущностей, остальные могут лишь просматривать. Также ограничен доступ к контенту для незалогиненых юзеров. Имеется функция восстановления пароля по  email. 
После запуска проекта необходимо перейти по:
http://127.0.0.1:8000/
Доступы для админа:
login: admin
password: admin
В моем приложении есть функция поиска по названию новости, добавление новости в избранное, так же есть количество просмотров новостей.
Имеются тестовые данные, для того, чтобы их загрузить: ./manage.py loaddata db.json

Эндпоинты API для возможности получения незалогиненым пользователям в JSON-формате списков категорий и статей:
1) Категории: http://127.0.0.1:8000/api/categories/
2) Статьи: http://127.0.0.1:8000/api/articles/
